# مراجعة نظام عروض الأسعار والعقود

## الوضع الحالي

### 1. جدول الكميات (BOQ) ✅
- يحتوي على: البند، الوصف، الوحدة، الكمية
- يحتوي أيضاً على: unitPrice, totalPrice (يجب إزالتها أو تركها فارغة)
- **التوصية**: جدول الكميات يجب أن يكون كميات فقط بدون أسعار

### 2. عروض الأسعار (Quotations) ✅
- totalAmount: المبلغ الأصلي من المورد
- approvedAmount: المبلغ المعتمد بعد التفاوض (موجود بالفعل!)
- status: pending, accepted, rejected, expired
- **الحالة**: يوجد حقل approvedAmount لكن لا يوجد حالة "قيد التفاوض"

### 3. العقود (contractsEnhanced) ✅
- contractAmount: قيمة العقد
- supervisionPercentage: نسبة الإشراف
- totalContractValue: إجمالي قيمة العقد (القيمة + النسبة)
- **الحالة**: يدعم نسبة الإشراف

## المطلوب

### إضافة ميزة التفاوض
1. إضافة حالة "negotiating" لعروض الأسعار
2. إضافة حقل negotiationNotes لملاحظات التفاوض
3. إضافة حقل negotiatedAmount للسعر بعد التفاوض
4. واجهة لإدخال نتيجة التفاوض

### ربط السعر بعد التفاوض بالعقد
- عند إنشاء عقد، استخدام negotiatedAmount إن وجد
- وإلا استخدام totalAmount الأصلي

### تفعيل الانتقال للخطوات التالية
- بعد الاعتماد المالي → التنفيذ
- بعد توقيع العقد → تحويل الطلب لمشروع
