# ملخص بوابة تمام للعناية بالمساجد

## نظرة عامة
بوابة تمام هي نظام متكامل لإدارة المساجد تابع لجمعية عمارة المساجد (منارة)، يهدف إلى تحسين تجربة المستفيدين وتبسيط العمليات الإدارية من خلال منصة رقمية موحدة.

## البرامج التسعة

| # | البرنامج | الوصف |
|---|----------|-------|
| 1 | بنيان | بناء مسجد جديد |
| 2 | دعائم | استكمال المساجد المتعثرة |
| 3 | عناية | الصيانة والترميم |
| 4 | إمداد | توفير تجهيزات المساجد |
| 5 | إثراء | سداد فواتير الخدمات |
| 6 | سدانة | خدمات التشغيل والنظافة |
| 7 | طاقة | الطاقة الشمسية |
| 8 | مياه | أنظمة المياه |
| 9 | سقيا | توفير ماء الشرب |

## الأدوار والصلاحيات (9 أدوار)

1. **المدير العام (Super Admin)** - صلاحيات كاملة على النظام
2. **مدير النظام (System Admin)** - المهام الإدارية والتقنية
3. **مكتب المشاريع (Projects Office)** - مراجعة واعتماد الطلبات
4. **الفريق الميداني (Field Team)** - الزيارات الميدانية والتقارير الفنية
5. **فريق الاستجابة السريعة (Quick Response)** - الطلبات العاجلة
6. **الإدارة المالية (Financial Management)** - الميزانيات والمدفوعات
7. **مدير المشروع (Project Manager)** - الإشراف على التنفيذ
8. **الاتصال المؤسسي (Corporate Communication)** - التواصل وقياس الرضا
9. **طالب الخدمة (Service Requester)** - المستفيد النهائي

## مراحل الطلب (7 مراحل)

1. تقديم الطلب
2. المراجعة الأولية
3. الزيارة الميدانية
4. التقييم الفني
5. التقييم المالي
6. التنفيذ
7. الإغلاق

## البنية التقنية

### الواجهة الأمامية (Frontend)
- React 19
- TypeScript
- Tailwind CSS 4
- shadcn/ui
- Wouter (routing)
- TanStack Query

### الواجهة الخلفية (Backend)
- Node.js 22
- Express 4
- tRPC 11
- Drizzle ORM

### قاعدة البيانات
- MySQL/TiDB

### الأمان
- JWT (JSON Web Tokens)
- bcrypt
- HTTPS/TLS

## هيكل قاعدة البيانات

### جداول المستخدمين
- users: بيانات المستخدمين
- employees: بيانات الموظفين
- password_reset_tokens: رموز إعادة تعيين كلمة المرور

### جداول المساجد
- mosques: بيانات المساجد

### جداول الطلبات
- mosque_requests: الطلبات الرئيسية
- request_attachments: مرفقات الطلبات
- request_comments: تعليقات الموظفين
- request_history: سجل تغييرات الحالة

### جداول التقارير
- field_visit_reports: تقارير الزيارات الميدانية
- quick_response_reports: تقارير الاستجابة السريعة
- final_reports: التقارير الختامية

### جداول المشاريع
- projects: المشاريع المعتمدة
- project_phases: مراحل التنفيذ
- contracts: عقود المقاولين
- payments: الدفعات المالية

### جداول التمويل
- donation_opportunities: فرص التبرع
- suppliers: الموردين والمقاولين
- quotations: عروض الأسعار
- quantity_schedules: جداول الكميات

### جداول الإشعارات
- notifications: الإشعارات المرسلة

### جداول الإعدادات
- categories: التصنيفات الرئيسية
- category_values: قيم التصنيفات
- brand_settings: إعدادات الهوية البصرية
- brand_logos: الشعارات
- brand_colors: الألوان
- homepage_customization: تخصيص الصفحة الرئيسية

### جداول التدقيق
- audit_logs: سجل التدقيق الكامل

## التكامل مع الخدمات الخارجية

1. **البريد الإلكتروني (SMTP)** - إرسال الإشعارات
2. **الرسائل النصية (SMS)** - الإشعارات العاجلة
3. **خرائط Google** - تحديد مواقع المساجد
4. **التخزين السحابي (S3)** - تخزين الملفات والصور
