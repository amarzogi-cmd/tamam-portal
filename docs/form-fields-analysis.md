# تحليل حقول نماذج الإدخال - بوابة تمام

## تصنيف البرامج حسب متطلبات المسجد

### المجموعة 1: برنامج بنيان (لا يتطلب مسجد مسجل)
- **برنامج بنيان** (بناء مسجد جديد)
- لا يتطلب وجود مسجد مسجل في النظام لأنه طلب بناء مسجد جديد

### المجموعة 2: البرامج التي تتطلب مسجد مسجل (7 برامج)
- **برنامج دعائم** (استكمال المساجد المتعثرة)
- **برنامج عناية** (الصيانة والترميم)
- **برنامج إمداد** (توفير تجهيزات المساجد)
- **برنامج إثراء** (سداد فواتير الخدمات)
- **برنامج سدانة** (خدمات التشغيل والنظافة)
- **برنامج طاقة** (الطاقة الشمسية)
- **برنامج مياه** (أنظمة المياه)

### المجموعة 3: برنامج سقيا (يتطلب مسجد مسجل)
- **برنامج سقيا** (توفير ماء الشرب)
- له حقول خاصة مختلفة عن المجموعة 2

## الحقول الخاصة بكل برنامج

### برنامج بنيان (بناء مسجد جديد)

| الحقل | النوع | الخيارات |
|-------|------|----------|
| هل هناك أرض مخصصة لبناء مسجد؟ | اختيار | نعم / لا |
| ملكية الأرض | اختيار | الأرض ملك لوزارة الشؤون الإسلامية / الأرض مملوكة لجهة حكومية / الأرض خاصة |
| مساحة الأرض بالمتر المربع | رقم | - |
| هل هناك متبرع لبناء المسجد؟ | اختيار | نعم يوجد متبرع لبناء المسجد بكامل مرافقه / نعم يوجد متبرع للتبرع بجزء من المبلغ / لا يوجد متبرع |
| كم مبلغ التبرع المتاح | رقم | يظهر إذا كان هناك متبرع |
| مقترحات توفير تكاليف البناء | نص طويل | يظهر إذا لا يوجد متبرع |
| مقترحات إيجاد أرض للبناء | نص طويل | يظهر إذا لا توجد أرض |
| أقرب مسجد لكم | نص | - |
| كم يبعد عنكم | نص | - |
| استعداد لتأسيس فريق تطوعي | اختيار | نعم / لا |
